# Обучение DBT

Активация виртуальной среды.
```bash
.venv\\scripts\\activate   
```
Переход в папку проекта
```bash
cd test_dbt
```

Генерация документации
```bash
dbt docs generate
```
Публикация документации
```bash
dbt docs serve
```
